Notes on Hoare Logic

1. Introduction

Hoare Logic is a logic for progamming languages. It is a logic to verify how programs are calculating. It is a logic to express what those programs are doing.

For example, we can imagine a C program which adding two integers. 

int add(int x, int y) {
    return x + y;
}

Then, add(2,3) returns $5$, add(-100, 100) returnds $0$.
